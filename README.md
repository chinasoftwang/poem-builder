poem-builder
============

a poem builder by javascript in IDE(Webstorm,Dreamweaver,Eclipse,Aptana,NetBeans,Visual Studio).

现代诗器，作诗机.
采用.js .html格式写诗，中文命名变量，一句诗即一个变量名，充分利用IDE的自动提示和自动完成功能！
